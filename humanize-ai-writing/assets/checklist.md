# Pre-ship checklist

Run before delivering any prose. One unchecked box = rewrite that sentence.

- [ ] **1. Vocabulary**: none of: delve, tapestry, testament, underscore,
  pivotal, intricate, meticulous, garner, vibrant, bolstered, enduring, interplay,
  crucial, valuable, exemplify, renowned, groundbreaking, boasts, nestled, realm,
  embark, leverage, elevate, unlock, harness, robust, seamless, holistic, myriad,
  cornerstone, paradigm, synergy. No openers: additionally, moreover, furthermore,
  notably, importantly. No "in today's world", "in the heart of", "rich cultural
  heritage", "it's important to note", "it's worth noting".
- [ ] **2. No fake significance**: no "stands as", "serves as a testament",
  "pivotal moment", "indelible mark", "setting the stage", "turning point",
  "evolving landscape", "deeply rooted".
- [ ] **3. No -ing padding**: no trailing "…, highlighting/reflecting/
  contributing to…" clauses that add no fact.
- [ ] **4. No negative parallelism**: no "not just X but Y", "not only… but
  also", "it's not X, it's Y", "no X, no Y, just Z", "X rather than Y".
- [ ] **5. No forced triples**: list length matches the real number of points.
- [ ] **6. Punctuation**: **no em dashes** (used comma / period / colon / parens
  instead); straight quotes only; almost no bold; sentence-case headings; no emoji
  as formatting; no inline-header bold lists.
- [ ] **7. No promo tone**: no "commitment to", "cutting-edge", "world-class",
  "passionate about", "we're excited to", "diverse array".
- [ ] **8. Plain copulas**: used "is/are/has", not "serves as / stands as /
  represents / features / maintains / offers / boasts".
- [ ] **9. No weasel / notability padding**: every "experts say / studies show /
  observers have cited / some critics argue" is named or cut; no media-coverage
  lists; no "active social media presence".
- [ ] **10. No hollow conclusion / formula**: no "In conclusion / Overall /
  Ultimately / In summary"; no "Despite its… faces challenges… Future Outlook";
  no future-speculation.
- [ ] **11. Rhythm**: sentence lengths vary; repetition allowed over synonym-
  cycling.
- [ ] **12. No artifacts**: no oaicite, contentReference, turn0search0,
  utm_source=, curly quotes, stray markdown.

## Final pass
- [ ] Facts, numbers, names unchanged from source.
- [ ] Didn't overcorrect, no real information deleted just to purge a word.
- [ ] Reads like a knowledgeable person wrote it, not a brand.
