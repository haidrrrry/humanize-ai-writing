# Pre-ship checklist

Run before delivering any prose. One unchecked box = rewrite that sentence.

## The 10 rules
- [ ] **1. Vocabulary** — no delve, tapestry, testament, underscore, pivotal,
  intricate, meticulous, garner, vibrant, nestled, boasts, leverage, foster,
  robust, seamless, holistic, myriad, cornerstone, realm, embark, elevate.
- [ ] **2. No fake significance** — no "stands as," "serves as a testament,"
  "pivotal moment," "indelible mark," "rich history/tapestry."
- [ ] **3. No -ing padding** — no trailing "…, highlighting/reflecting/
  contributing to…" clauses that add no fact.
- [ ] **4. No negative parallelism** — no "not just X but Y," "it's not X, it's
  Y," "more than just," "X rather than Y."
- [ ] **5. No forced triples** — list length matches the real number of points.
- [ ] **6. Punctuation** — ≤1 em dash per few paragraphs; straight quotes; almost
  no bold; sentence-case headings.
- [ ] **7. No promo tone** — no "commitment to," "cutting-edge," "world-class,"
  "passionate about," "we're excited to."
- [ ] **8. No weasel words** — every "experts say / studies show / many argue"
  is named or cut.
- [ ] **9. No hollow conclusion** — no "In conclusion / Overall / Ultimately";
  no future-speculation wrap-up.
- [ ] **10. Rhythm** — sentence lengths vary; repetition allowed over synonym-
  cycling.

## Final pass
- [ ] No leftover markup artifacts (oaicite, contentReference, :::, turn0search0).
- [ ] Facts, numbers, names unchanged from source.
- [ ] Length appropriate — not padded.
- [ ] Reads like a knowledgeable person wrote it, not a brand.
