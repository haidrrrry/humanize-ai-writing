# LinkedIn launch post

AI writing has a fingerprint. Once you see it, you can't unsee it.

The same words show up everywhere: delve, tapestry, underscore, leverage. The same shapes too. "It's not just a tool, it's a movement." "Stands as a testament to innovation." A trailing "in conclusion" that says nothing.

Readers notice. It makes good ideas read like filler.

So I built a free, open-source fix. It's a set of rules, based on Wikipedia's "Signs of AI writing" page, that makes any AI tool write like a person instead.

How it works:
- Claude: install it as a skill. It runs on "humanize this."
- ChatGPT, Gemini, Grok, Kimi: paste one prompt into custom instructions.
- Cursor, Antigravity, Copilot, and other coding tools: drop one rules file in your project.

It also ships a small command-line scanner that reads your draft and flags every tell with a suggested fix, so you can check your own writing before you publish.

No signup, no paywall. MIT licensed.
